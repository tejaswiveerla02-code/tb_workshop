# 🎓 Gate Pass Management System

A simple web-based system that allows students to request gate passes and enables gatekeepers/moderators to verify and manage requests efficiently.
---
## ✨ Features

### 👩‍🎓 Student Panel

* Submit gate pass request
* Enter student ID and reason
* View date & time of request

### 🚪 Gatekeeper Panel

* Enter Pass ID to verify request
* Check pass details instantly

### 🛡️ Moderator Panel

* View all requests
* Approve or Reject passes
* Add remarks
* Track request status (Pending / Approved / Rejected)

## ▶️ How to Run


2. Open the project folder
3. Run in browser:
